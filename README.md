# Predictive Modeling Using Machine Learning

## 📌 Project Overview
This project focuses on building a predictive machine learning model using supervised learning techniques. The model is trained on the Titanic dataset to predict passenger survival based on features such as age, gender, class, and more.

---

## 🎯 Objective
To build and evaluate a machine learning model that can accurately predict outcomes using classification algorithms.

---

## 📊 Dataset
The dataset used is the Titanic dataset which includes:
- Passenger information
- Survival status (target variable)

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 🤖 Machine Learning Model
- Random Forest Classifier
- Train-Test Split approach
- Supervised Learning technique

---

## 🧹 Data Preprocessing
- Handled missing values
- Removed unnecessary columns (Name, Ticket, Cabin)
- Converted categorical data into numerical format
- Encoded variables using label encoding and one-hot encoding

---

## 📈 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- ROC Curve

---

## 🚀 Results
- Model achieved good accuracy on test data
- Visualized performance using confusion matrix and ROC curve
